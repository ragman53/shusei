    - Real Android device connected via USB
    - USB debugging enabled
    - Device has Android 7.0+ (API 24+)

    ## Steps

    ### 1. Build and Deploy
    ```bash
    dx serve --android
    ```
    Wait for app to compile and deploy to device.

    ### 2. Launch and Navigate
    - App should launch automatically
    - If not, tap app icon on device
    - Verify Library screen appears

    ### 3. Test Load Demo PDF
    - Look for orange "Load Demo PDF" button on Library screen
    - Tap the button
    - Verify:
      - [ ] Button responds to tap
      - [ ] No crash or ANR
      - [ ] Metadata dialog appears (or PDF imports directly)

    ### 4. Test PDF Processing
    - If OCR processing starts:
      - [ ] Progress indicator shows
      - [ ] UI remains responsive (can scroll, tap)
      - [ ] No OOM crash
      - [ ] Processing completes without errors

    ### 5. Verify Results
    - [ ] PDF appears in library list
    - [ ] Can open and view the PDF
    - [ ] Extracted text is readable (if OCR ran)
